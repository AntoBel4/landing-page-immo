⚡ Installation des 6 Assistants IA

  • Ouvrez le dossier 01_Assistants_IA Choisissez le premier fichier (ex: 01_Generateur_Consentement.txt).
  • Allez sur ChatGPT (Version Plus recommandée) Cliquez sur Explorer les GPTs > Créer (Create).
  • Copiez-Collez les données Dans l'onglet Configurer, copiez le Nom, la Description et les Instructions exactes du fichier texte.
  • Désactivez la navigation Web Pour éviter les hallucinations, décochez "Web Browsing" sauf si indiqué.
  • Sauvegardez en "Privé" Cliquez sur Créer/Mettre à jour et choisissez "Moi uniquement". Ces outils sont votre avantage concurrentiel, ne les partagez pas.
